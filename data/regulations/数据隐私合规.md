# 数据隐私合规审查指引

## 适用范围
- 涉及个人信息处理的合同
- 跨境数据传输
- 数据共享协议

## 审查要点
1. 是否明确数据处理目的和范围
2. 是否获得数据主体同意
3. 是否有数据安全保护措施
4. 是否符合数据本地化要求

## 风险等级判定
- **高风险**：跨境传输 + 无安全保障
- **中风险**：国内共享 + 保障措施不完整
- **低风险**：已获同意 + 安全措施完整

## 借贷担保合规审查指引（支撑型业务板块）

### 适用范围
- 对外担保合同
- 借贷担保、抵押、质押合同
- 金融机构借款相关合同
- 保险合同

### 审查要点
1. 担保合同是否关联主合同
2. 担保金额是否在授权范围内
3. 是否经过法务和财务双重审查
4. 担保条款是否存在无限责任风险
5. 借贷合同是否上传审批文件
6. 抵押/质押物权属是否清晰

### 风险等级判定
- **高风险**：大额担保 + 无限责任条款 + 未关联主合同
- **中风险**：担保金额较大 + 审批程序不完整
- **低风险**：担保金额合理 + 已关联主合同 + 审批完整
